= 通用部分

docbook.type::
    描述:::
        记录了 docbook 文档顶级元素的名称。
    可能的取值:::
        article::::
            *【支持】* 文章，表示当前是一篇文章
        book::::
            *【支持】* 书，表示当前的 docbook 文档是一本书
        bookset::::
            *【不支持】* 丛书，表示当前的 docbook 文档是一部丛书
        slider::::
            *【不支持】* 幻灯片，表示当前的文档是一份幻灯片
